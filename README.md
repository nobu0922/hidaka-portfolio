# ヒダカ｜WebコーダーのポートフォリオLP

PC・スマホ両対応（レスポンシブ対応）で、軽量かつSEOを意識した構造になっています。

- アニメーション・jQueryを用いたインタラクション
- 納品実績50件以上の経験を反映
